Public addItemWillAddObserver
addItemWillAddObserver(o: (view: View,container: View) => void)

Добавляет наблюдатель: элемент будет добавлен.

Наблюдатель принимает два параметра: view и container.

Элементы в ListView хранятся в контейнерах. Поэтому при добавлении слушатель получает два View элемента. Благодаря этому слушателю, Вы можете кастомизировать контейнер.

Имя обсервера: itemWillAdd
