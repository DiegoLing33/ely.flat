Public get
get(guard?: T)

Возвращает значение или guard если значение null.

Данный метод никогда не возвращает значение null. В случае, если значение прослушиваемого параметра null или undefined, возвращает guard значение.

Параметры :
Название Тип данных Опционально
guard T Да
Возвращает : T | null

// Создаем прослушиваемый параметр let prop = new ObservableProperty();

// Отображаем в консоль "защищенное" значение (с флагом guard) console.log( prop.get( "test" ) ); // test
