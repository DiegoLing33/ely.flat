Static timeCodeToVars
timeCodeToVars(timeCode: number)

Преобразует временной код в части: дни, часы, минуты, секунды. Такая технология может быть полезна для создания таймеров.

Использование:

  • Сначала необходимо получить разницу веремни, например, используя вычитание;
  • Полученное значение может быть трансформировано через этот метод.
